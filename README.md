## Quick start

```bash
# Claude Code (recommended)
cp -r agency-agents/* ~/.claude/agents/

# Then in any session:
# "Activate Frontend Developer mode and help me build a React component"
```

Also works with Cursor, Aider, Windsurf, Gemini CLI, and OpenCode:

```bash
./scripts/convert.sh    # Generate integration files
./scripts/install.sh    # Auto-detect and install
```
